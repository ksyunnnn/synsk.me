# Vision

> Working Backwards: 完成した未来から逆算して、今何を作るべきかを明確にする

## Press Release

### Headline

**synsk.me** — 一緒に奏でて、あなたの中にあるものを形にする

### Subheadline

課題を抱える人、これから何かを始める人のための、実験的な共創の場。
技術とデザインのセッションを通じて、まだ見えない可能性を引き出す。

### Summary

synsk.me は、小橋俊介のポートフォリオであり、共創の入り口です。

10年以上のプロダクト開発経験を通じて培った「形にする力」と「引き出す力」を、課題を抱える人・社会に向けて開いています。地方・行政課題、孤立した家族への支援、コミュニティ形成といった領域に関心を持ち、技術とデザインを道具として、まだ言葉にならないアイデアを一緒に形にします。

### Problem

「やりたいこと」や「解決したい課題」はあるけれど、どう形にすればいいかわからない。

エンジニアやデザイナーに相談しても、要件がまとまっていないと話が進まない。でも、要件をまとめるには一緒に考えてくれる人が必要——そんな鶏と卵の状況に陥っている人がいます。

### Solution

synsk.me は、まだ形になっていないアイデアを一緒に探り、即興的なセッションを通じて形にしていくプロセスを提供します。

- **対話から始める**: 要件定義の前に、対話を通じて本質を探る
- **実験しながら見つける**: 完璧を求めず、小さく作って試す
- **技術とデザインで具現化する**: 言葉にならないものを、動くプロダクトとして形にする

### Quote（作り手として）

> 「10年以上、様々なスタートアップでプロダクトを作ってきました。その中で気づいたのは、最も良いものが生まれるのは、要件が固まってからではなく、『何を作るべきか』から一緒に考えたときだということ。だから、要件がなくても相談できる相手でありたい。」

### How It Works

1. **コンタクト**: サイトまたはFacebookメッセンジャーからご連絡ください
2. **対話**: まずは話を聞かせてください。要件がなくても大丈夫です
3. **セッション**: 一緒に考え、実験し、小さなプロトタイプを作ります
4. **形にする**: 必要に応じて、本格的な開発・デザインへ

## FAQ

### External FAQ（訪問者向け）

**Q: これは何ですか？**

A: 小橋俊介のポートフォリオであり、共創のパートナーを探すための場です。過去の実績を見せるだけでなく、これから一緒に何かを作りたい人との接点を作ることを目的としています。

**Q: 誰のためのものですか？**

A: 課題を抱えている人、社会課題に取り組んでいる人、これから何かを始める人、投資・支援を考えている人。特に、地方・行政課題、孤立した家族への支援、コミュニティ形成に関心がある方との相性が良いです。

**Q: 他のポートフォリオと何が違いますか？**

A: 多くのポートフォリオは「こんなことができます」という実績のショーケースです。synsk.me は「一緒に何ができるか」を探る入り口として設計しています。実績は、共創の可能性を示すための参考情報です。

**Q: 依頼には何が必要ですか？**

A: 要件定義や予算が決まっていなくても大丈夫です。「こんなことをやりたい」「こんな課題がある」という想いだけで十分です。対話を通じて一緒に整理していきます。

### Internal FAQ（自分向け）

**Q: なぜこのプロジェクトを始めたのか？**

A: 自分の経験と技術を、おもしろいと思える課題に使いたいから。様々なプロジェクトに関わる中で、より深く、長く、社会的インパクトのある仕事に関わりたいと考えるようになった。

**Q: 成功とは何か？どう測るか？**

A: 定量的には「このサイトを通じて始まったプロジェクトの数」。定性的には「一緒に作った人が、自分の可能性を実感できたか」。

**Q: やらないことは何か？**

A: 要件が固まった状態での単なる「作業」としての開発。対話なく進めるプロジェクト。価値観が合わない仕事。正社員としての参加。
