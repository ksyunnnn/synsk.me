## 変化の軸

### 1. 技術スタック

```
2014-2016: Rails, Vagrant, GitLab（学習期）
    ↓
2019-2020: Vue.js, Nuxt.js, React, Firebase
    ↓
2021-2022: TypeScript, GraphQL, Apollo Client, Figma
    ↓
2023-2025: React/TypeScriptエコシステムに集中
```

**変化**: 幅広い技術探求から、TypeScript/Reactを軸にした専門性へ

### 2. 働き方

```
2014-2016: 会社員（マーケ→エンジニア転向中）
    ↓
2019-2020: フリーランス確立、東京固定拠点
    ↓
2021-2022: ワーケーション実践（沖縄1ヶ月）
    ↓
2023-2025: 場所に縛られない柔軟なスタイル確立
```

**変化**: 会社員 → フリーランス → 場所に縛られない働き方

### 3. 自己認識

```
2014-2016: 「スピード速いが精度が低い」（課題認識）
    ↓
2019-2020: 「まだまだ修行していきたい」（成長志向）
    ↓
2021-2022: 「型定義をしないことは自分が何をしているのかわからず実装しているのと同義」（自信と専門性）
    ↓
2023-2025: 「おっさんが落ち着いて見えるのは元気がないから」（成熟・等身大）
```

**変化**: 課題認識 → 成長志向 → 専門家としての自信 → 成熟した自己受容

### 4. 社会への視点

```
2014-2016: メディア倫理批判、「ゴミサイトはなくなれ」
    ↓
2019-2020: イジメの本質（進化心理学的視点）への共感
    ↓
2021-2022: 問題解決の本質「どの問題なら許容できるか」
    ↓
2023-2025: SNS炎上文化批判「正論でも攻撃は正当化できない」
```

**変化**: 感情的な批判 → 構造的理解 → 哲学的洞察 → 成熟した批判的視点

## 一貫しているもの（不変の軸）

### 1. 関西弁・カジュアルな語調
10年間一貫して「〜やけど」「〜やな」「〜やろ」を使用。東京在住でも変わらない。

### 2. 社交性・人とのつながり重視
- 2016: 勉強会主催「恵比寿でもくもく」
- 2019: 「軽率に誘ってほしい」
- 2022: 沖縄で色んな人とつながる
- 継続的に友人との交流を大切にする

### 3. ユーモアと自虐
時期を問わず、自虐的なユーモアを交えたツイートスタイル。

### 4. 学習意欲・成長志向
2016年の「シリコンバレー挑戦」から2022年の「バックエンドへの苦手意識克服」まで、常に新しい領域に挑戦し続ける姿勢。

### 5. 率直さ・オープンさ
感情をストレートに表現し、私生活もオープンに共有するスタイルは一貫。

## 思考の深化

### 信頼論
```
2016: 「自分を信じない人がすごく信頼できる」
  ↓ 深化
2021: 「そのままでええんかどうかは未来の自分にしかわからん」
```

### 問題解決
```
2016: 課題をひたすら解決しようとする姿勢
  ↓ 深化
2022: 「問題の存在を完全に消し去ることはできず、どの問題なら許容できるか」
```

### 成長観
```
2016: 成長への強い渇望
  ↓ 深化
2025: 「人は歳とったぐらいで成長なんてしません」（成長神話への懐疑）
```

---

## 総括

10年間のツイートから見える変化は、**「渇望する若者」から「成熟した専門家」への成長軌道**。

しかし、関西弁、ユーモア、社交性、学習意欲といった**核となる部分は一貫**しており、表面的なスキルや働き方が変わっても、人としての本質は変わっていない。

特筆すべきは、単なる技術的成長だけでなく、**思考の深化**が見られること。問題解決、信頼、成長といったテーマへの洞察が年々深まっている。
