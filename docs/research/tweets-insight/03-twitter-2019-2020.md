## 3. デザインへの深い考察

### グラフィック vs UI デザイン
> 「グラフィックや静的なサイトのデザインとUIのデザイン、デザインするコンテンツやデータが予め決まっているかいないかという点で、デザインに必要な能力明らかに違いそう」

### デザインの4原則について
> 「デザインの4原則とか言ってるフェーズは、表示するコンテンツやデータが決まっている(あるいは高い解像度で想定できている)ときにやっと適用できるもの」
