## Overview

写真家/ディレクター/デザイナーのポートフォリオサイト。宮城県仙台市在住。
PRINCIPLES.md で参考として挙げられたサイトであり、「余白」「息づき」「対話」の原則を体現している。

## Typography

### Font Stack

| 用途 | フォント | 配信方法 |
|------|----------|----------|
| 日本語 | FontPlus経由のカスタムフォント | [FontPlus](https://fontplus.jp/) |
| 英語 | サンセリフ（Grotesk系と推測） | FontPlus |
| ヒーロー見出し | セリフ体（イタリック） | FontPlus |

**実装**: Tailwind CSS + FontPlus Webフォントサービスのハイブリッド構成

### Type Scale

サイト独自の命名規則を使用（`text-r24` 等の "r" prefix）

| Token | 推定サイズ | 用途 |
|-------|------------|------|
| `text-r44` | 44px | フッター大リンク |
| `text-r38` | 38px | ヒーロー見出し |
| `text-r24` | 24px | セクション見出し |
| `text-r20` | 20px | サブ見出し |
| - | **20-24px** | **ステートメント本文（日本語）** ※特徴的 |
| `text-16` | 16px | 説明文、プロフィール |
| `text-14` | 14px | ラベル、メタ情報 |
| `text-11.5` | 11.5px | キャプション |
| `text-11` | 11px | 最小テキスト |

**特徴**: ABOUTページのステートメント（「この世界に存在できる時間は...」）は **20-24px 相当** の大きなサイズ。1行あたり25-30文字程度。これは一般的な本文サイズ（16px）よりかなり大きく、「余白 over 密度」を強く体現している。

### Line Height

| Token | 値 | 用途 |
|-------|-----|------|
| `leading-180` | 1.8 | 日本語本文 |
| `leading-170` | 1.7 | 英語本文 |
| `leading-129` | 1.29 | 見出し |
| `leading-100` | 1.0 | タイトル |

**特徴**: 日本語本文の行間が非常に広い（1.8）。呼吸感・余白を重視。

### Letter Spacing

| Class | 用途 |
|-------|------|
| `tracking-[.01rem]` | ロゴ、ナビゲーション |
| `tracking-tight` | 大見出し |

### Font Weight

| 用途 | ウェイト |
|------|----------|
| サイト名「YOSHIYUKI KUBO」 | Regular (400) |
| プロジェクトタイトル | Medium/Semi-bold (500-600) |
| 本文 | Regular (400) |
| フッターリンク `(1616)` | **Thin/Light (100-300)** |
| ナビゲーション | Regular (400) |

**特徴**: フッターの外部リンクは極細ウェイト + 大サイズで、静かだが存在感がある

### Text Transform

| 要素 | 変換 |
|------|------|
| サイト名 | `uppercase` |
| プロジェクトタイトル | `uppercase` |
| ナビゲーション | `uppercase` |
| パンくず | `uppercase` |
| 本文・場所名 | なし（通常ケース） |

## Layout & Spacing

### Navigation

- **位置**: 右下固定（デスクトップ・モバイル共通）
- **項目**: HOME / ABOUT のみ
- **スタイル**: 大文字、letter-spacing 広め

### Content Structure

```
[ロゴ: 左上]                           [ナビ: 右下]

              [メインコンテンツ: 中央]
              - 写真カード（傾いた配置）
              - プロジェクトタイトル + 場所
```

### Card Layout

- 写真は中央配置、微妙に傾いている（rotation）
- タイトルは画像左下に直接配置（背景なし）
- ホバーで「VIEW」表示

## PRINCIPLES.md との対応

| 原則 | サイトでの体現 |
|------|----------------|
| **余白 over 密度** | line-height 1.8、letter-spacing 広め、コンテンツ量抑制、カード間の余白 |
| **息づき over 装飾** | 極細ウェイト、静かなトランジション、赤はアニメーションのみに限定 |
| **対話 over 展示** | 写真は個別カード形式（グリッド一覧ではない）、詳細ページへの導線 |

## synsk.me への示唆

### 採用を検討すべき点

1. **大きめの本文サイズ（18-20px）**: ステートメント的な文章に余白感を与える
2. **広い行間（1.7-1.8）**: 日本語本文の可読性と余白感
3. **大文字 + letter-spacing**: ナビゲーション、ラベルの静かな存在感
4. **極細ウェイトの活用**: 大きなテキストに Light/Thin を使う手法
5. **固定ナビゲーション**: 右下配置のミニマルなナビ

### 採用しない / 検討が必要な点

1. **FontPlus**: 有料サービス。Google Fonts で代替可能か検討
2. **写真中心のレイアウト**: synsk.me はテキスト/プロジェクト中心になる可能性
3. **赤のブランドカラー**: synsk.me はモノクロファーストを採用済み
